jQuery Team Select

A jQuery plugin for adding team members (agents) to an input.

Usage

1. Include jQuery and the plugin.

          <script src="path/to/jquery.js"></script>
          <script src="path/to/jquery.team-select.js"></script>

2. Include the plugin styles

<link rel="stylesheet" href="path/to/team-select.css">

3. Examples

<textarea id="teamSelector"></textarea>
$(document).ready(function() {
            $('#teamSelector').teamSelect();
          });




4. Example with custom font-size

<input id="teamSelector2" >
$(document).ready(function() {
        $('#teamSelector2').teamSelect(8);
      });


5. To get selectd ids, in the console.

$( "#teamSelector" ).val()
